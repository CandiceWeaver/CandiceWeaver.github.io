Mapty

About the project:Mapty is an app where you can log and track your running and cycling workouts. It renders your current location on to the screen so you can see your workouts visually as well. Furthermore, Mapty will remember your workouts so they will still appear next time you use the app.

What I learned:This project was part of a JavaScript course on Udemy. I learned a lot of new skills with this project. For example, I learned how to use API and read their documentation, how to use geolocation to get the user's current location and, although it's not best practice, how to save data in the local storage.
